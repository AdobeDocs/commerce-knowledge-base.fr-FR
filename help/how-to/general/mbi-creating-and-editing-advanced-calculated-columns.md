# MBI : création et modification de colonnes calculées avancées

Les [utilisateurs administrateurs](https://experienceleague.adobe.com/fr/docs/commerce-business-intelligence/mbi/administrator/user-mgmt/user-management) peuvent créer et modifier certaines colonnes calculées dans Data Warehouse Manager. Ces colonnes calculées vous permettent de configurer les rapports que vous recherchez pour répondre aux questions commerciales clés.

Pour obtenir la liste des colonnes calculées que vous pouvez créer dans Data Warehouse Manager, ainsi que des conseils pour vous assurer de les configurer correctement, consultez les articles suivants de notre guide de l’utilisateur :

* [Création de colonnes calculées](https://experienceleague.adobe.com/fr/docs/commerce-business-intelligence/mbi/analyze/warehouse-manager/creating-calculated-columns)
* [Types de colonnes calculées](https://experienceleague.adobe.com/fr/docs/commerce-business-intelligence/mbi/analyze/warehouse-manager/calc-column-types)
* [Comprendre et évaluer les relations entre les tables](https://experienceleague.adobe.com/fr/docs/commerce-business-intelligence/mbi/analyze/warehouse-manager/table-relationships)
* [Création de chemins d’accès pour les colonnes calculées](https://experienceleague.adobe.com/fr/docs/commerce-business-intelligence/mbi/analyze/warehouse-manager/create-paths-calc-columns)

## J&#39;ai encore besoin d&#39;aide !

Si vous avez besoin d’aide pour créer une colonne calculée normale ou souhaitez créer ou modifier une colonne calculée avancée, [contactez notre équipe](https://experienceleague.adobe.com/fr/docs/support-resources/adobe-support-tools-guide/adobe-commerce-support/adobe-commerce-help-center-user-guide#submit-ticket) et incluez les éléments suivants :

* Qu&#39;est-ce que tu essaies de faire ?
* Sur quelle table avez-vous besoin que la ou les nouvelles colonnes soient créées ?
* Quels sont les chemins entre cette table et les autres ?
* Quelles colonnes doivent être utilisées comme entrées dans la ou les nouvelles colonnes calculées ?
* Quelle logique faut-il suivre pour créer la ou les nouvelles colonnes ? Ex : MIN, MAX, etc.

Si vous ne savez pas comment répondre à l&#39;une de ces questions, ne vous inquiétez pas ! Faites de votre mieux, et nous vous aiderons à le parcourir.
