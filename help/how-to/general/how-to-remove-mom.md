# Comment supprimer Magento Order Management (MOM)

1. Désactivez l’intégration MOM en suivant les étapes décrites dans [Désactiver ou activer l’intégration](https://commerce-docs.github.io/oms-documentation-archive/integration/connector/#disable-or-enable-the-integration).
1. Désactivez le module MOM en suivant les étapes décrites dans [Désinstaller les modules](https://experienceleague.adobe.com/docs/commerce-operations/installation-guide/tutorials/uninstall-modules.html?lang=fr).
1. Pour extraire des données de commande complètes, nous proposons l’API . Pour en savoir plus, consultez [Référentiel de commandes](https://commerce-docs.github.io/oms-documentation-archive/specifications/#magento.sales.order_repository) dans notre Adobe | Documents Magento OMS relatifs aux informations de commande (order_repository). Utiliser la [section des spécifications](https://commerce-docs.github.io/oms-documentation-archive/specifications/#services) dans notre Adobe | Documents OMS Magento pour utiliser d’autres API afin d’extraire différents types d’informations.
