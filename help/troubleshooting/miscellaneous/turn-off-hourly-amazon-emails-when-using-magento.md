# Désactiver les e-mails Amazon horaires lors de l’utilisation d’Adobe Commerce

## Question

Je reçois un e-mail d’Amazon toutes les heures. Le module effectue-t-il ceci ?

## Réponse

Oui. Afin de maintenir une synchronisation continue des systèmes, Adobe Commerce récupère les rapports d’Amazon et les compare à votre catalogue Adobe Commerce, puis vous envoie des e-mails de notification basés sur ces rapports toutes les heures. Vous avez la possibilité de refuser ces e-mails de rapport de notification si vous le souhaitez.

## Pour vous désinscrire de ces e-mails :

Suivez ces instructions pour [Mettre à jour les préférences de notification sur le site Amazon](https://sellercentral.amazon.com/gp/help/external/G871).
